# Song-O-Gram_DelAngelPonceAlejandrina_IP

Song-o-Gram is an application whose operation consists on graphic interpretation of different musical pieces, complementing the task of musical education, appreciating works graphically and understanding music as something beyond sound, being this object of interactive creations and even a bridge towards the realization of digital artistic works.

Music application made in Processing using Sound and Minim libraries. Homework for Interactive Programmation subject.
